## **Getting Started** üöÄ

Before running this project, make sure you have the following installed:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/) (optional, only if you want to use npm for static server)
- [npm](https://www.npmjs.com/) (optional)

## **Installation Instructions**

1. **Clone the Repository in your terminal**:

```bash
git clone https://github.com/Victoriia-Melnyk/Poems_generator_project.git
```

2. **Open the cloned project in your IDE (e.g., VSCode)**

3. **Run with Go Live (recommended for VS Code):**
   - Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code.
   - Open the project in VS Code.
   - Right-click on `index.html` and select "Open with Live Server" or click the "Go Live" button at the bottom of the editor.
   - The project will open in your browser at an address like `http://127.0.0.1:5500/index.html`.
